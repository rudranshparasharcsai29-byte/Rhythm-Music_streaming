## 🚀 Quick Start (< 5 Minutes)

### 1. Initialize
```bash
cd backend
node setup.js
```

### 2. Install
```bash
npm install
```

### 3. Configure
```bash
# Edit .env with your PostgreSQL connection
nano .env
# DATABASE_URL=postgresql://user:password@localhost:5432/rhythm_db

# Generate and migrate
npx prisma generate
npx prisma migrate dev --name init
```

### 4. Run
```bash
npm run dev
```

### 5. Test
```bash
curl http://localhost:3000/health
```

## 🆘 Troubleshooting

| Issue | Fix |
|-------|-----|
| npm install fails | `npm cache clean --force` |
| Cannot find @prisma/client | `npx prisma generate` |
| DB connection error | Check DATABASE_URL and PostgreSQL |
| Port in use | Change PORT in .env |
| TypeScript errors | `npm run type-check` |
| Files not organized | Run `node setup.js` again |
